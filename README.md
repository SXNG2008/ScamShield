# ScamShield

ScamShield is a behavioral fraud interruption system that detects
victims of digital scams in real time and intervenes before money
is lost.

## Problem

India loses over ₹1 lakh crore annually to digital fraud.

Existing systems focus on detecting scammers.

ScamShield focuses on detecting victim behavior.

## Features

- Call monitoring
- OTP event detection
- Remote-access app detection
- Fraud confidence scoring
- Full-screen intervention

## Tech Stack

- Android Studio
- Kotlin
- Accessibility Service
- TelephonyManager
- Text-to-Speech
